Welcome to giFramework

giFramework is a PHP micro framework that provides everything you need to code small to medium web applications. What differs from other frameworks ? It doesn't give in to bandwagon.

  • whole source code can be read in less than 30 minutes
  • no dependencies
  • no command lines tools required
  • very small memory footprint and quick execution time

While providing :

  • local & production environments
  • clever and elegant routing
  • incredibly simple database abstraction
  • MVC capabilities
  • ultrafast cache
  • output formating
  • strong security/authentication
  • translation engine
  • discrete debuging tools
$ cd /var/www/
$ git clone https://github.com/AnnoyingTechnology/giframework2.1

Point your webserver/vhost to /var/www/giframework2/public/ If you're under apache, that's it. If you are under anything else, redirect (rewrite) everything except CSS/ JS/ FONTS/ IMG/ to /
